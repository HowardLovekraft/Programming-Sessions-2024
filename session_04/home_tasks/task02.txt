TASK

Написать функцию, которая будет фильтровать значения последовательности (=списка) и возвращать новый список, из которого вычищены все значения нечислового типа.

INPUT_DATA

Список значений с рандомными типами данных

OUTPUT_DATA

Список значений только с целочисленными данными (будем считать, что bool не является целочисленным типом)

TESTS
# пускай функция называется foo, вы называйте её как хотите

TEST 1
>>> foo([1, 4, [3, 4], "3", "29", -13, True, 3.14)
[1, 4, -13]

TEST 2
>>> foo([-13, "345", 42.42, tuple(3), "28", 0, False, 14, True, 1])
[-13, 0, 14, 1]
