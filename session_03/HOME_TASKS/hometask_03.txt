TASK

Программе через стандартный поток ввода (т.е. input() ) подается число - квадрат какого-то числа.
Программа должна вывести "следующий" квадрат.

Если введено число, которое нельзя получить путем возведения в квадрат целого числа, программа должна распечатать сообщение об этом. +креатив.

Комментарий:
Проверить то, что число можно получить, возведя в квадрат другое целое число, можно с помощью такой проверки:
int(num**0.5)**2 == num

Такая проверка вернет True, если число num можно получить путем возведения в квадрат другого целого числа.
Работает проверка за счет погрешности в записи вещественных чисел в памяти компьютера.


INPUT DATA
Число в стандартном потоке ввода - квадрат числа (допустим N)

OUTPUT DATA
Квадрат следующего числа (т.е. квадрат числа N+1)


EXAMPLES:
Example 1.
>>> 121
144.0 # квадрат числа 12

Example 2.
>>> 625
676.0 # квадрат числа 26

Example 3.
>>> 114 # не является квадратом целого числа
-1
