TASK

Вводится число N - количество значений, которое потребуется считать с клавиатуры (с помощью input() )
Затем в программу вводится N чисел - каждое в отдельной строке.
Требуется вывести сумму положительных значений (т.е. >= 0).
Если положительных значений среди N чисел нет - вывести в консоль 0.

Комментарий.
Считать число с клавиатуры == привести строку (input() возвращает строку) к типу int. Это можно сделать с помощью int()


INPUT DATA
Число N - количество значений, которое потребуется считать с клавиатуры
N раз - целое число, каждое вводится в отдельной строке

OUTPUT DATA
Число - сумма положительных значений среди введенной последовательности


EXAMPLES:
Example 1.
>>> 4 # длина вводимой последовательности
>>> 5
>>> 1
>>> 1
>>> -1
7

Example 2.
>>> 3   # длина вводимой последовательности
>>> -13
>>> -42
>>> 0
0

Example 3.
>>> 5   # длина вводимой последовательности
>>> -13
>>> 8
>>> 14456
>>> -12432
>>> 11
14475
